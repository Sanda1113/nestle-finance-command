# Nestle Finance Command Center (AI-Powered 3-Way Match)

An automated, enterprise-grade financial reconciliation engine that leverages AI to perform 3-way matching between Vendor Invoices and Purchase Orders (PO). The system features a multi-portal architecture designed for both Suppliers and Finance Teams.

## 🚀 Key Features

* **AI Document Extraction**: Powered by **Mindee V5 Custom Models** to extract line items, totals, and metadata from unstructured PDFs/Images.
* **Automated 3-Way Match**: Real-time discrepancy detection between Invoice and PO totals with automated approval routing.
* **Multi-Portal Architecture**:
    * **Supplier Portal**: Secure document upload and instant reconciliation status.
    * **Finance Review Queue**: A management dashboard for manual overrides (Approve/Reject) on flagged discrepancies.
    * **Real-time Analytics**: High-level throughput metrics, auto-approval rates, and total value processed.
* **Enterprise UI**: Responsive dashboard built with **Tailwind CSS v4** featuring a seamless Dark/Light mode toggle.
* **Immutable Ledger**: Automated database logging to **Supabase (PostgreSQL)** for a secure audit trail.

## 🛠️ Tech Stack

### Frontend
* **React.js**: Modern component-based UI.
* **Tailwind CSS v4**: High-performance "CSS-first" styling engine.
* **Axios**: Asynchronous API communication.
* **Vercel**: Production frontend hosting.

### Backend
* **Node.js & Express**: Robust middleware API.
* **Mindee SDK**: Advanced AI/OCR extraction engine.
* **Supabase SDK**: Real-time PostgreSQL database management.
* **Railway**: Production backend hosting.

### Infrastructure & CI/CD
* **GitHub Actions**: Automated CI/CD pipeline for clean builds.
* **Environment Security**: Protected API keys via Railway/Vercel secrets.
